# Employee Management System

This project is a microservices-based employee management system designed
to manage employees, payroll, and other HR-related services.

## Architecture

- Microservices architecture
- REST APIs

## Services

- employee-management
- auth-service (future)

## Technologies

- Go
- Gin
- PostgreSQL
- Docker
- Swagger (OpenAPI)

## How to run

See each service README for setup instructions.
